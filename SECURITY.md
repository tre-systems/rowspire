# Security Policy

## Supported Code

Security fixes target the current `main` branch and the application deployed at [rowspire.com](https://rowspire.com). Historical commits and local modifications are not supported releases.

## Reporting a Vulnerability

Use [GitHub private vulnerability reporting](https://github.com/tre-systems/rowspire/security/advisories/new). Do not disclose a suspected vulnerability in a public issue.

Include the affected URL or commit, reproduction steps, impact, and any suggested mitigation. You should receive an acknowledgement within three working days and a status update within seven working days. Please allow a reasonable remediation window before public disclosure.
